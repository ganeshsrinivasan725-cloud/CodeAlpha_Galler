# Responsive Image Gallery

Features:
- Responsive grid layout with CSS
- Hover effects and smooth transitions
- JavaScript lightbox with next/prev navigation and keyboard support
- Category filters and simple image filters (grayscale, sepia)

How to run:
1. Open `index.html` in a browser (no server required).

Notes:
- Images use placeholder photos from picsum.photos. Replace `img` `src` values with your assets as needed.
- To expand: add thumbnails, lazy-loading, pagination or tags.
